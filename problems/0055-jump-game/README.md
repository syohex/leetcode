# memo

- stackを用意. index候補が格納される
- 初めに初期位置 0 を stackに設定
- ループ stackが空になるまで
  - stackから一つ取り出す(pop)
  - その indexの移動量を調べ 0なら ループ継続
  - この indexを調査済みとする(-1を設定)
  - 移動量が 0以上の場合(nとする)
    - current index + i(1 <= i <= n)を次の候補として stackに追加する
    - ただし移動先の移動量が 0もしくは調査済みの場合は候補外とする

## 補足

- 最後の候補を減らす処理がない場合は時間超過でアウトだった
